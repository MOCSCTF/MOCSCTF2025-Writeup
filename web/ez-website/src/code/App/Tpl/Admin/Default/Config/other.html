分页设置

前台搜索页面分页大小 默认为：20条，根据显示器屏幕高度设置适当值。
前台分页条可见页码个数 默认为：10条，根据显示器屏幕宽度设置适当值。
后台分页大小 默认为：20条，根据显示器屏幕高度设置适当值。此外，当记录数很多时，设置更大的分页大小，更容易删除所有记录
后台分页条可见页码个数 默认为：30条，根据显示器屏幕宽度设置适当值。
