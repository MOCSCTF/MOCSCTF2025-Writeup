订单设置

货币符号 设置页面显示的货币符号：人民币符号是￥，美元符号是$
订单编号前缀 常设为大写字母，订单编号格式：前缀+年月日时分秒+4位随机数字！如：YN201601181201580001
订单提交权限     
每页显示订单数
订单提交是否启用验证码      启用后，提交订单时必须输入验证码
订单商品金额满多少免运费 0：表示不免运费，请设置为整数
积分换算比例 1元等于  积分 必须设置为整数。当用积分兑换商品时，积分兑换现金比例。如设置为10，表示1个积分可以抵扣0.1元
发货后多少天自动收货 如果模板没有购物功能，请设置为"禁止自动收货"

邮件通知设置

是否邮件通知      启用后，客户提交订单后，会发送邮件通知。
接收邮件帐号 多个邮箱以英文逗号隔开，留空为[管理首页]->[联系方式]设置的电子邮件
邮件标题
邮件内容 标题和内容均支持变量： {$OrderTime}表示下单时间   {$OrderNumber}表示订单编号   {$TotalOrderPrice}表示订单总额  

订单短信通知设置

是否短信通知      启用后，客户提交订单后，会发送短信通知。
手机号码 注意：仅支持一个手机号码
短信通知模板
模板内容举例：您有新的订单：订单编号【{$OrderNumber}】，订单金额【{$TotalOrderPrice}】，下单时间【{$OrderTime}】，请及时处理！
设置好模板后，通常需要提交到短信服务商，审核以后才能使用。 支持模板变量： {$OrderTime}表示下单时间   {$OrderNumber}表示订单编号   {$TotalOrderPrice}表示订单总额
