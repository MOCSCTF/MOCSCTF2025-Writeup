会员功能设置

是否启用会员功能      如果禁用，将不能使用所有会员相关功能，如注册、登录、找回密码等
是否允许会员注册     
是否审核注册会员               
注册激活邮件
选择“注册邮件激活”时有效，内容支持HTML。 {$MemberName}表示当前会员名称   {$WebName}表示网站名称   {$Activate}表示激活链接  
是否启用注册验证码      启用后，用户注册时必须输入验证码。如果禁用，可能会受攻击，建议开启
是否启用登录验证码      启用后，用户登录时必须输入验证码。如果禁用，可能会受攻击，建议开启
是否审核用户添加的文章      启用后，用户添加的文章在审核后才能显示

手机验证码短信模板
模板内容举例：您的验证码是：{$Code}。请不要把验证码泄露给其他人。   设置好模板后，通常需要提交到短信服务商，审核以后才能使用。 支持模板变量：{$Code}表示当前手机注册验证码
